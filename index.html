Set Size

How to use

Choose a color from the right column, under the 'Choose Color' title. Then just draw on the grid.

The Brightness Tool can be used to modify the brightness value of the grid squares.

The size of the grid can be set by clicking the 'Set Size' buttons under the 'Set Size' title.

To clear the grid simply click on the 'Clear' button.
